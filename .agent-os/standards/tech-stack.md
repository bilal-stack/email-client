# Default Tech Stack

This is the Agent OS *default* stack for a similar future project. The locked stack for *this* project lives in `.agent-os/product/tech-stack.md` and takes precedence.

- Next.js 15 (App Router) + TypeScript strict
- Auth.js v5
- Prisma + Postgres
- shadcn/ui + Tailwind v4
- TanStack Query + Zustand
- React Hook Form + Zod
- Vitest + Playwright + MSW
- Biome
- npm
- `@anthropic-ai/sdk` with prompt caching
- Inngest for background jobs
- Server-Sent Events for one-way real-time
