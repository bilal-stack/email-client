# Graph Provider (lite)

Second concrete `IEmailProvider` adapter using `@microsoft/microsoft-graph-client`. Implements list / get / send / reply / archive / trash / markRead / setLabels / search / syncDelta against Microsoft Graph for Office 365 mailboxes. Refactors `lib/providers/auth.ts` to dispatch token refresh by provider (adds a Microsoft refresh path that **persists rotated refresh tokens** — MS rotates on every refresh, unlike Google). Extends `error-mapping.ts` to translate Graph `410 Gone` on a stale `@odata.deltaLink` into `AuthError` ("Sync delta expired — reconnect required") in parallel with Gmail's stale-historyId 404. Synthetic-label mapping: Inbox/Sent/Drafts/Deleted-Items folders + `isRead` flag → `INBOX`/`SENT`/`DRAFT`/`TRASH`/`UNREAD` strings; user labels round-trip through Graph **categories**. Delta sync per `/me/mailFolders/inbox/messages/delta`; new Inngest cron mirrors `gmail-sync`. **No schema changes** — Gmail-provider's tables are already provider-neutral. **No UI changes** — the inbox / thread / composer / search / labels surfaces shipped earlier consume `IEmailProvider` and must light up unchanged for Microsoft users.
